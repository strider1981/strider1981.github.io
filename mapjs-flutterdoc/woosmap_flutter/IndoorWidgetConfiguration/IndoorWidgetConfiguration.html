Default constructor. Creates IndoorWidgetConfiguration object.

autocompleteWithDistance - Sort POIs in autocomplete based on the distance.

avoidZones - Areas to avoid while routing. Useful when certain areas of the venues are restricted or under maintenance. Expected format evel;lat,lng;lat,lng;lat,lng...|level;lat,lng;lat,lng;lat,lng...

baseFilter - Filter the map and the autocomplete to display/search only labels and icons of POIs which are matching the filters.

directionsMode - Set directions mode/routing profile.

navigationMode - Sets whether to enable/disable navigation mode.

ui - Set the custom colors of the indoor widget. Object of IndoorWidgetConfigurationUI.

units - Set units to use to show distance in directions itinerary. Of UnitSystem. Default is UnitSystem.metric
